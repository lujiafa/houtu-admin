## 一、描述
这是一套权限管理系统，实现菜单+角色+用户+岗位+组织等进行权限一体化控制管理的系统，基本满足企业级管理平台权限控制要求。

## 二、技术架构
采用前后端分离，一定程度提升开发效率、优化用户体验、增强代码可维护性、提升性能与扩展性等等。
### 2.1 前端
前端采用Vue3+ElementPlus+iconify+Pinia+I18n+vite+typescript等等框架语言组合实现。代码结构清晰，可维护性强。

### 2.2 后端
后端Java版本为JDK17+，SpringBoot采用3.5.x以上版本，结合SpringSecurity、MybatisPlus、Redis等等，实现部分自定义增强校验。项目结构清晰，可维护性强。
